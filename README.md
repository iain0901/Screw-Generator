# OpenSCAD 螺絲元件生成器

這是一個基於 OpenSCAD 的參數化螺絲元件生成工具。本工具提供完整的中文介面，方便使用者快速生成各種規格的螺絲元件，適合用於 3D 列印。

## 🛠 可生成的元件類型

- 螺栓
  - 多種頭型可選
  - 可自訂驅動方式
  - 支援標準規格
- 螺母
  - 標準六角螺母
  - 蝶形螺母（適合手動調整）
- 墊圈
  - 可自訂內外徑
  - 可調整厚度
- 螺桿
  - 全螺紋或部分螺紋
  - 可選擇倒角處理

## ✨ 主要特色

### 螺絲頭型選擇
- 六角頭（標準工業規格）
- 套筒頭（適合狹小空間）
- 圓頭套筒（美觀實用）
- 沉頭套筒（適合平面安裝）

### 驅動方式選擇
- 內六角套筒（高扭力）
- 十字槽（通用型）
- 一字槽（簡單實用）

### 參數化設計
- 完全可自訂尺寸
- 符合工業標準
- 支援 ISO262 公制標準
- 特殊尺寸客製化

## 💻 使用指南

### 基本需求
- OpenSCAD 軟體（[下載連結](https://openscad.org/downloads.html)）
- 基本的 3D 列印知識
- 了解基本的螺絲規格

### 使用步驟
1. 下載並安裝 OpenSCAD
2. 將 `screw_generator.scad` 檔案下載到本地
3. 使用 OpenSCAD 開啟檔案
4. 在參數面板中選擇：
   - 元件類型
   - 尺寸規格
   - 特殊需求設定
5. 預覽生成結果
6. 匯出 STL 檔案進行列印

## 📐 預設參數說明

### 通用預設值
- 螺絲直徑：8mm
- 螺距：2mm
- 螺紋角度：45°

### 建議參數（ISO262 標準）
| 螺絲規格 | 螺距 | 建議螺母外徑 |
|---------|------|------------|
| M3      | 0.5  | 5.5mm      |
| M4      | 0.7  | 7mm        |
| M5      | 0.8  | 8mm        |
| M6      | 1.0  | 10mm       |
| M8      | 1.25 | 13mm       |
| M10     | 1.5  | 16mm       |

## 🖨️ 3D 列印建議

### 列印參數
- 列印速度：30-40mm/s
- 層高：0.1-0.2mm
- 填充率：≥75%
- 外圍層數：≥3

### 材料建議
- PLA：適合一般用途
- PETG：較好的強度和耐溫性
- ABS：適合機械應用場合

### 品質優化建議
1. 使用低速列印提高精度
2. 降低列印溫度減少溢料
3. 增加冷卻改善螺紋品質
4. 先進行測試列印調整參數

## 📝 常見問題解答

### Q1：螺絲和螺母配合太緊？
- 調整螺母內徑參數
- 檢查列印溫度是否過高
- 確認列印機校準狀況

### Q2：螺紋品質不佳？
- 降低列印速度
- 增加冷卻
- 調整層高設定

### Q3：強度不足？
- 增加填充率
- 嘗試使用更適合的材料
- 調整列印方向
